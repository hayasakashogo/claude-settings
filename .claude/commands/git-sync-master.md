masterブランチにチェックアウトし、リモートの最新変更をプルします。

以下の手順を Bash ツールで実行してください：

1. 現在のブランチを `git branch --show-current` で確認する
2. 未コミットの変更がある場合は `git status --short` で確認し、ユーザーに警告する
   - 変更がある場合は「未コミットの変更があります。処理を中断します。」と伝えて終了する
3. `git checkout master` で master ブランチにチェックアウトする
4. `git pull origin master` でリモートの最新変更を取得する
5. 完了後に現在のブランチ名と最新のコミットハッシュ・メッセージを表示する
